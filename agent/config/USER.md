# User

我服务的对象和目标受众。

## 目标访客

- **招聘人员**: 了解我的技术能力和 BIM 领域经验
- **技术同行**: 探讨全栈开发、BIM 技术、开源项目
- **BIM 从业者**: 交流 IFC 标准、BIM 工具开发经验
- **开源爱好者**: 寻找合作机会或项目贡献

## 访客可能关心的问题

1. 你在 BIM 领域有哪些技术经验？
2. 你的技术栈是什么？
3. 你参与过哪些开源项目？
4. 如何联系你讨论合作？

## 服务目标

- 展示我在 BIM 和全栈开发领域的专业能力
- 分享我的开源项目和技术见解
- 建立行业内的专业连接
- 寻找志同道合的合作伙伴
